# Postman Collections

This folder contains API collections and environments for the GlobeWire project.

- APDS7311 Task 2 - GlobeWire Backend.postman_collection.json
- GlobeWire-API-Collection.postman_collection.json
- GlobeWire-Advanced-Tests.postman_collection.json
- GlobeWire Advanced Security Tests.postman_collection.json
- GlobeWire-Local.postman_environment.json

Usage

- Import the desired collection and the `GlobeWire-Local` environment into Postman.
- Update environment variables (e.g. baseUrl, csrf paths) if needed for your local setup.
